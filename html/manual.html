Инструкция по работе с расширением

Общий принцип работы

Для работы с данным расширением, Вам необходимо заполнить описание вашей задачи, установить время окончания задачи и наконец выставить сложность вашей задачи.

После того как вы выставите сложность задачи, произойдёт сохранение ваших данных для последующего использования.

По окончанию времени для выполнения задачи, программа оповестит вас звуковым сигналом.

Описание выбора сложности

Для выбора сложности Вам будет предоставлено три кнопки: "Легко", "Нормально" и "Сложно".

Кнопка "Легко" определяет задачи низкого уровня, которые можно выполнить в течении дня.

Кнопка "Нормально" определяет задачи среднего уровня, которые можно выполнить в течении нескольких дней и для решения данной задачи возможно придётся немного подумать.

Кнопка "Сложно" определяет задачи высокого уровня, которые можно выполнить также в течении нескольких дней и для решения данной задачи возможно придётся подумать гораздо больше чем над другими задачами.

Также на форме присутствует кнопка "Очистить" она убирает всю информацию о задаче.

Описание выбора окончания времени для решения задачи

Для выбора времени используется специальное поле, при нажатии на которое будет появляться календарь с выбором даты и времени.

Выбор времени для каждой задачи рекомендуется выбирать исходя из своих навыков и количества свободного времени, которое вы посвящаете работе.

Описание обратного отсчёта

Обратный отсчёт считает оставшееся время в зависимости от даты окончания работы над задачей.

"Разминка"

В разделе "Разминка" вас ожидают приятные и несложные упражнения для снятия напряжения и давления с ваших глаз.

Данные упражнения взяты с медицинских источников и имеют исключительно оздоровительные цели.

Для проведения разминки Вам необходимо всего лишь следовать инструкциям указнным в разделе.
